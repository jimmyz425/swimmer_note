# Swimming Session Template — Universal Slot System

> Created: 2026-05-17
> Self-contained reference for describing, comparing, and generating swim training sessions across all tiers.
> Aligned with SwimNote's USA Swimming club tier system (`TrainingTier`: Pre-Competitive, Bronze, Silver, Gold, Senior, National).

---

## Overview

This document defines a **canonical 10-slot session template** that can describe any swim practice — from a 6-year-old's first session to an Olympic taper workout. Every top club worldwide uses the same structural anatomy; what varies is the content within each slot.

The template enables:
- **Describing** any session using a consistent notation
- **Comparing** how different coaches, tiers, and purposes fill each slot
- **Generating** sessions by picking options from a catalog
- **Scaling** sessions up or down by swimmer tier

---

## The 10-Slot Template

Each session is composed of zero or more of these slots, in this fixed order:

| Slot ID | Slot Name | Required? | % of Session | Physiological Purpose |
|---|---|---|---|---|
| **A** | Warm-up | Always | 10-15% | Raise core temp, activate CNS, establish rhythm |
| **B** | Activation / Pre-set | Optional | 5-10% | Bridge warm-up to main work; ramp speed or technique focus |
| **C1** | Drill Block 1 | Usually | 10-15% | Technique focus area 1 (often stroke-specific) |
| **C2** | Drill Block 2 | Optional | 5-10% | Technique focus area 2 (different stroke or skill) |
| **D** | Kick Set | Optional | 5-10% | Leg strength, kick technique, aerobic contribution from legs |
| **E1** | Main Set 1 | Always | 30-50% | Primary training stimulus (aerobic/threshold/IM/etc.) |
| **E2** | Main Set 2 | Optional | 10-20% | Secondary stimulus (speed endurance, lactate, sprint, etc.) |
| **F** | Speed / Race Skills | Optional | 5-10% | Max velocity, starts, turns, breakouts, race-pace rehearsal |
| **G** | Pull Set | Optional | 5-10% | Upper-body endurance, stroke feel, aerobic without legs |
| **H** | Cool-down | Always | 5-10% | Flush metabolites, lower HR, promote recovery |

### Template Rules

- **Slots always appear in order** A -> B -> C1 -> C2 -> D -> E1 -> E2 -> F -> G -> H, but optional slots may be skipped.
- A **minimal session** has slots: A, C1, E1, H (warm-up, drills, main set, cool-down).
- A **maximal session** uses all 10 slots.
- **Slot names are fixed** -- any session from any coach can be mapped into this structure.
- For **Pre-Competitive (under 10)**, slot F is often replaced with a **Game/Relay** activity serving the same time slot.
- For **Gold+ sessions**, slot B (Activation) is nearly always present; for Pre-Competitive, it is often merged into A (Warm-up).

## Session Notation

Every session described using this template uses the following notation:

```
[A] Warm-up: 400 easy choice + 4x50 drill
[B] Activation: 4x25 build
[C1] Drill Block 1: 8x50 (25 drill/25 swim) freestyle
[C2] Drill Block 2: 8x50 (25 drill/25 swim) backstroke
[D] Kick Set: 4x50 kick with board
[E1] Main Set 1: 6x100 freestyle on interval
[E2] Main Set 2: 8x50 descend 1-4
[F] Speed/Race Skills: 6x25 sprint off blocks
[H] Cool-down: 200 easy
```

Skipped slots are simply omitted from the notation. This makes it trivial to compare how different coaches, tiers, and session purposes fill each slot.

### Tier Capability Summaries

**Pre-Competitive (1/2/3)** — Just transitioning from learn-to-swim. Developing water comfort, basic strokes, and enjoyment. Sessions are game-based and exploratory. No time standards required.

**Bronze (1/2/3)** — First-year competitive swimmers. Working toward B times and legal strokes. Building consistency across all four strokes. Time standard reference: B times.

**Silver (1/2/3)** — Developing aerobic engine. Working on A times, building volume tolerance. Time standard reference: B-A times.

**Gold (Sprint/Distance/Mixed)** — Senior age group swimmers. A-AA time standard range. Sprint: 50-200m event focus, race-pace specificity. Distance: 400m+ focus, aerobic capacity. Mixed: IM/versatile, balanced speed-endurance.

**Senior (Sprint/Distance/Mixed)** — Championship-level swimmers. AA-AAA time standards. Sprint: championship sprint focus, lactate tolerance. Distance: championship distance, endurance pacing. Mixed: IM/multi-event championship.

**National (Sprint/Distance/Mixed)** — Elite / national qualifier swimmers. AAAA / national cuts. Sprint: elite 50-200m specialist. Distance: elite 400m+ specialist. Mixed: elite IM/multi-event. 8-12+ practices per week.

### Tiered Targets in Technique Files

Individual technique markdown files (e.g., `butterfly-02-dolphin-kick-mechanics.md`) embed tiered performance targets using callout blocks like `> [!tip] Tiered Vertical Dolphin Kick Hold Times`. These use a four-level scheme parsed by `TechniqueMarkdownParser`:

| Tier Level | Maps to TrainingTier |
|---|---|
| Beginner | Pre-Competitive / Bronze |
| Intermediate | Silver |
| Advanced | Gold |
| Elite | Senior / National |

### Scaling Rules

- **Distance**: scale each rep distance to match the tier's volume range and round to the nearest standard pool distance (25, 50, 75, 100, 150, 200, etc.).
- **Intervals**: adjust send-off intervals proportionally to maintain the same effort level. Shorter reps for lower tiers, longer reps for higher tiers.
- **Rest**: maintain the same work-to-rest ratio across tiers.
- **Session count**: younger tiers have fewer sessions per week. See `PlanWeeklyDistanceTargets.swift` for `recommendedPracticesPerWeek` and `maximumPracticesPerWeek` by tier/sub-tier.
- **Equipment**: some options (paddles, parachutes, tempo trainers) are not appropriate for Pre-Competitive. The option descriptions note age restrictions.
- **Zone mix**: use tier guidance from `PlanTierGuidancePrompt` for training zones, intensity distribution, and focus areas per tier.
