## How to Use This Document

### Where These Fit in a Session

These drills are a **secondary drill set** — complementary to your regular (standard) drill work. They are not a replacement but an addition to explore new movement patterns and training methods.

| Order | Component | Purpose |
|---|---|---|
| 1 | Warm-up | Get blood flowing, loosen muscles |
| 2 | Standard drill set | Usual technique work (e.g., catch-up, fist drill, kick sets) |
| **3** | **Evidence-based drill set (pick one from below)** | **Skill exploration — the focus of this document** |
| 4 | Main set | Conditioning / fitness work |
| 5 | Cool-down | Active recovery |

Pick one drill per session and use it as the secondary set. Rotate through different drills across weeks to keep training varied and engaging.

### Intensity Zones (CSS-Based)

| Zone | CSS Pace Offset | HR% | Lactate | Description |
|---|---|---|---|---|
| **Z0** | CSS + 20-30s/100m | <60% | <1.5 mmol/L | Recovery — very easy, active rest |
| **Z1** | CSS + 10-15s/100m | 60-75% | 1.5-2.0 | Aerobic base — conversational |
| **Z2** | CSS + 5-10s/100m | 75-82% | 2.0-2.5 | Aerobic endurance — moderate, sustainable |
| **Z3** | CSS + 0-5s/100m | 82-88% | 2.5-3.5 | Tempo — "comfortably hard" |
| **Z4** | CSS to CSS - 2s/100m | 88-92% | 3.5-5.0 | Lactate threshold — hard |
| **Z5** | CSS - 3-6s/100m | 92-98% | 5.0-8.0 | VO2max — very hard |
| **Z6** | 50-100m race pace | 98-100% | 8-14+ mmol/L | Sprint — maximum effort |

## Freestyle Drills

### Drill F1: Tempo Ladder (Progressive Stroke Rate Build)
**Evidence:** Franken et al. (2023) — increasing stroke rate accelerated VO₂ kinetics; nearly doubled time to exhaustion.
**Base:** 11 items · 350m

| # | Reps | Distance | Intensity | Rest | Equipment | Description | Notes |
|---|---|---|---|---|---|---|---|
| 1 | 2 | 25m | Z1 | 15s | none | Easy swim at comfortable stroke rate | CSS + 10-15s/100m; loose, relaxed |
| 2 | 8 | 25m | Z3→Z6 | 20s | tempo | Build stroke rate progressively each 25m | Start Z3 (CSS pace); end Z6 (race pace); +2-4 BPM per rep |
| 3 | 1 | 100m | Z0 | — | none | Easy, focus on technique | CSS + 20-30s/100m |

**Level Adjustments:**

| Level | Item 2 Reps | Max Intensity | Rest | Notes |
|---|---|---|---|---|
| Beginner | 4 × 25m | Z3 | 30s | Focus on finding comfortable stroke rate; skip the sprint finish; use +2 BPM increments |
| Intermediate | 6 × 25m | Z5 | 20s | Build to threshold pace; +3 BPM per rep |
| Advanced | 8 × 25m | Z6 | 20s | Full progression CSS → race pace; +4 BPM per rep |
| Elite | 8 × 25m + 4 × 25m | Z6 | 15s | Do the 8-rep set, rest 60s, then 4 more at max sustainable rate; +4-6 BPM per rep |

**Progression (4 weeks):**
- Week 1: item 2 = 4 × 25m builds
- Week 2: item 2 = 6 × 25m builds
- Week 3: item 2 = 8 × 25m builds
- Week 4: item 2 = 8 × 25m builds, target +6 BPM from rep 1 to rep 8

### Drill F2: Roll Explorer (Rotation Angle Variations)
**Evidence:** Psycharakis & Sanders (2010) — optimal shoulder roll ~30-45° for middle-distance; faster swimmers roll LESS over distance.
**Base:** 4 items · 250m

| # | Reps | Distance | Intensity | Rest | Equipment | Description | Notes |
|---|---|---|---|---|---|---|---|
| 1 | 2 | 25m | Z1 | 15s | none | Easy freestyle, moderate rotation | CSS + 10-15s/100m |
| 2 | 4 | 25m | Z2 | 15s | none | Each rep: different rotation angle | See angle table below; CSS + 5-10s/100m |
| 3 | 1 | 25m | Z2 | 15s | none | Natural — find your optimal | Comfortable pace |
| 4 | 1 | 50m | Z0 | — | none | Easy recovery swim | CSS + 20-30s/100m |

**Item 2 — Rotation angles:**

| Rep | Rotation | Focus |
|---|---|---|
| 25m #1 | Minimal (~10°) | Feel the flat position, shoulder engagement only |
| 25m #2 | Moderate (~30°) | Controlled roll, find the efficient range |
| 25m #3 | Exaggerated (~55°) | Explore the upper limit, feel the drag |
| 25m #4 | Natural (self-selected) | Integrate what you learned into your natural stroke |

**Level Adjustments:**

| Level | Item 2 Reps | Rest | Notes |
|---|---|---|---|
| Beginner | 2 × 25m (#2, #4 only) | 30s | Start with just moderate and natural; add the extremes later |
| Intermediate | 4 × 25m | 20s | All 4 angles; video yourself to compare |
| Advanced | 2 rounds × 4 × 25m | 15s | Complete the block twice; second round, try to swim faster at each angle |
| Elite | 3 rounds × 4 × 25m | 15s | Third round, add paddles to exaggerate the rotation feel |

> [!warning] Research: Faster swimmers roll LESS over 200m. Use this drill to find your efficient range, not to maximize rotation.

### Drill F3: Differential Practice (Variable Practice Set)
**Evidence:** Soleimani et al. (2025) — differential learning (no repeated movement) significantly outperformed traditional repetitive training.
**Base:** 12 items · 550m

| # | Reps | Distance | Intensity | Rest | Equipment | Description | Notes |
|---|---|---|---|---|---|---|---|
| 1 | 2 | 25m | Z1 | 15s | none | Easy freestyle | CSS + 10-15s/100m |
| 2 | 1 | 25m | Z2 | 15s | none | Normal, focus on smooth catch | Baseline feel |
| 3 | 1 | 25m | Z2 | 15s | none | High elbow catch only (EVF emphasis) | Forearm vertical |
| 4 | 1 | 25m | Z2 | 15s | none | 3-stroke breathing pattern | Bilateral control |
| 5 | 1 | 25m | Z3 | 15s | none | Build stroke rate first 15m, hold last 10m | Tempo surge |
| 6 | 1 | 25m | Z2 | 15s | none | Exaggerated body rotation | Explore range |
| 7 | 1 | 25m | Z2 | 15s | none | Minimal body rotation | Flat swimming |
| 8 | 1 | 25m | Z2 | 15s | fist | Fist drill | Forces forearm catch |
| 9 | 1 | 25m | Z2 | 15s | none | Single-arm right only (left at side) | Balance + rotation |
| 10 | 1 | 25m | Z2 | 15s | none | Single-arm left only (right at side) | Balance + rotation |
| 11 | 1 | 25m | Z6 | 30s | none | Sprint — max effort, clean technique | Full power, race pace |
| 12 | 1 | 100m | Z0 | — | none | Easy recovery | CSS + 20-30s/100m |

**Level Adjustments:**

| Level | Items | Intensity Cap | Rest | Notes |
|---|---|---|---|---|
| Beginner | Items 2, 3, 4, 6, 8, 12 (skip 5, 7, 9, 10, 11) | Z3 | 30s | Focus on technique variety; skip sprint; skip single-arm if balance is poor |
| Intermediate | Items 2-12 (all) | Z6 | 20s | Full set; increase item 11 effort gradually |
| Advanced | Items 1-12 + repeat items 5-11 as round 2 | Z6 | 15s | Second round, push intensity higher; aim to improve feel from round 1 |
| Elite | Items 1-12 + 2 more rounds, each faster | Z6 | 15s | Each round, increase pace 5% while maintaining variety; use tempo trainer to lock in rate |

**Progression:** Do this drill once per week. Each week, try to make every length feel distinctly different — the variety is the point.

### Drill F4: Build & Hold (Race Finish Simulation)
**Evidence:** Franken et al. (2023) — controlled stroke rate manipulation as physiological overload strategy.
**Base:** 3 items · 600m

| # | Reps | Distance | Intensity | Rest | Equipment | Description | Notes |
|---|---|---|---|---|---|---|---|
| 1 | 2 | 50m | Z1 | 15s | none | Easy swim, build to moderate | CSS + 10-15s/100m |
| 2 | 6 | 50m | Z4→Z5 | 30s | none | Build SR first 35m, hold max rate last 15m | Z4 (LT) → Z5 (VO2max); race-finish simulation |
| 3 | 1 | 100m | Z0 | — | none | Easy recovery | CSS + 20-30s/100m |

**Level Adjustments:**

| Level | Item 2 Reps | Build / Hold Split | Max Intensity | Rest | Notes |
|---|---|---|---|---|---|
| Beginner | 4 × 50m | Build 40m / Hold 10m | Z4 | 45s | Short hold phase; focus on maintaining form, not speed |
| Intermediate | 6 × 50m | Build 35m / Hold 15m | Z5 | 30s | Standard set; hold must be genuine race-pace effort |
| Advanced | 6 × 50m | Build 30m / Hold 20m | Z5 | 25s | Shorter build, longer hold — harder race simulation |
| Elite | 8 × 50m | Build 25m / Hold 25m | Z6 | 20s | Long hold at race pace; add paddles on reps 5-8 for overload |

**Progression (4 weeks):**
- Week 1: item 2 = 4 × 50m, build over 25m / hold 25m
- Week 2: item 2 = 4 × 50m, build over 35m / hold 15m
- Week 3: item 2 = 6 × 50m, build over 35m / hold 15m
- Week 4: item 2 = 6 × 50m, build over 40m / hold 10m (harder hold phase)

### Drill F5: Constraints Circuit
**Evidence:** Brackley et al. (2020) — constraints-led approach shapes movement without explicit instruction.
**Base:** 5 items · 400m

| # | Reps | Distance | Intensity | Rest | Equipment | Description | Notes |
|---|---|---|---|---|---|---|---|
| 1 | 2 | 25m | Z1 | 15s | none | Easy freestyle | CSS + 10-15s/100m |
| 2 | 1 | 50m | Z2 | 15s | fist | Fist drill | Forces forearm catch |
| 3 | 1 | 50m | Z2 | 15s | none | Catch-up drill | Glide timing + extension |
| 4 | 1 | 50m | Z2 | 15s | none | Single-arm alternating | Rotation + balance |
| 5 | 1 | 50m | Z2 | 30s | none | Sculling | Feel for water |

**Level Adjustments:**

| Level | Items 2-5 Rounds | Distance Per Item | Rest | Notes |
|---|---|---|---|---|
| Beginner | 1 round | 25m each | 30s | Half distance; focus on technique, not speed; skip single-arm if balance is poor, substitute with kick-on-side drill |
| Intermediate | 1 round | 50m each | 15s | Standard set; maintain clean technique on each constraint |
| Advanced | 2 rounds | 50m each | 15s | Second round, increase pace to Z3; focus on transferring constraint feeling to free swim |
| Elite | 3 rounds | 50m each | 10s | Round 1 = Z2, Round 2 = Z3, Round 3 = Z4; between rounds, do 50m free swim and check if the constraint feeling transfers |

**Progression:** Repeat items 2-5 as a circuit, 2-3 rounds. Each round, focus on a different aspect (e.g., round 1 = technique, round 2 = speed, round 3 = endurance).

## Breaststroke Drills

### Drill BR1: Tempo Ladder (Progressive Stroke Rate Build)
**Evidence:** Franken et al. (2023)

> [!note] Breaststroke "building stroke rate" = progressively shortening the glide phase while maintaining clean pull-kick timing.

**Base:** 3 items · 350m

| # | Reps | Distance | Intensity | Rest | Equipment | Description | Notes |
|---|---|---|---|---|---|---|---|
| 1 | 2 | 25m | Z1 | 15s | none | Easy breaststroke, normal glide | CSS + 10-15s/100m |
| 2 | 8 | 25m | Z3→Z6 | 20s | none | Progressively shorten glide phase each 25m | Start Z3 (long glide); end Z6 (minimal glide, sprint tempo) |
| 3 | 1 | 100m | Z0 | — | none | Easy recovery | CSS + 20-30s/100m |

**Level Adjustments:**

| Level | Item 2 Reps | Max Intensity | Rest | Notes |
|---|---|---|---|---|
| Beginner | 4 × 25m | Z3 | 30s | Focus on clean transitions between pull-kick; don't eliminate glide completely |
| Intermediate | 6 × 25m | Z5 | 20s | Progress from moderate glide to minimal glide by rep 6 |
| Advanced | 8 × 25m | Z6 | 20s | Rep 8 = race-pace tempo with almost no glide |
| Elite | 8 × 25m + 4 × 25m | Z6 | 15s | After 8 reps, rest 60s, then 4 more at max sprint tempo with zero glide |

**Progression (4 weeks):**
- Week 1: item 2 = 4 × 25m, focus on smooth transitions
- Week 2: item 2 = 6 × 25m, focus on maintaining kick power with shorter glide
- Week 3: item 2 = 8 × 25m, focus on clean pull timing
- Week 4: item 2 = 8 × 25m, hold same final tempo across all reps

**Evidence:** Seifert & Carmigniani (2023) — coordination parameters vary by event and swimmer profile.
**Base:** 7 items · 250m

|---|---|---|---|---|---|---|---|

## Butterfly Drills

### Drill FL1: Tempo Ladder (Progressive Stroke Rate Build)
**Evidence:** Franken et al. (2023)

> [!note] Butterfly's two-kick cycle means stroke rate changes affect kick timing. Build rate while maintaining enter-kick / push-kick rhythm.

**Base:** 3 items · 350m

| # | Reps | Distance | Intensity | Rest | Equipment | Description | Notes |
|---|---|---|---|---|---|---|---|
| 1 | 2 | 25m | Z1 | 15s | none | Easy butterfly or easy swim | CSS + 10-15s/100m |
| 2 | 8 | 25m | Z3→Z6 | 25s | tempo | Build stroke rate progressively each 25m | Start Z3; end Z6 (race pace); +2-4 BPM per rep; maintain two-kick rhythm |
| 3 | 1 | 100m | Z0 | — | none | Easy recovery | CSS + 20-30s/100m |

**Level Adjustments:**

| Level | Item 2 Reps | Max Intensity | Rest | Notes |
|---|---|---|---|---|
| Beginner | 2 × 25m (butterfly) + 2 × 25m (easy swim) | Z3 | 45s | Alternate butterfly and easy swim; focus on maintaining two-kick rhythm; skip sprint |
| Intermediate | 4 × 25m butterfly | Z5 | 25s | All butterfly; build to threshold pace; +3 BPM per rep |
| Advanced | 8 × 25m | Z6 | 25s | Full progression; +4 BPM per rep; every rep must maintain clean two-kick cycle |
| Elite | 8 × 25m + 4 × 25m | Z6 | 20s | After 8 reps, rest 60s, then 4 more at max sprint rate; add resistance band for overload |

**Progression (4 weeks):**
- Week 1: item 2 = 4 × 25m (or easy swim between reps)
- Week 2: item 2 = 6 × 25m
- Week 3: item 2 = 8 × 25m
- Week 4: item 2 = 8 × 25m, target +6 BPM from rep 1 to rep 8

## Level Adjustment Summary

Quick reference for scaling any drill:

| Parameter | Beginner | Intermediate | Advanced | Elite |
|---|---|---|---|---|
| **Reps** | 50-75% of base | 100% of base | 100-125% of base | 125-150% of base |
| **Max Intensity** | Z3 max | Z6 | Z6 | Z6 (precisely calibrated) |
| **Rest** | +50% vs base | Base | Base or -25% | Base or -50% |
| **Distance per item** | 25m | 50m | 50m | 50m |
| **Rounds** | 1 | 1 | 2-3 | 3-4 |
| **Complexity** | Simplified items | Full set | Full set + repeats | Full set + overload variations |
| **Focus** | Technique, feel | Technique + pace | Pace + transfer | Race-specific precision |
